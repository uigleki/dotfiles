* emacs 常用按键绑定

- ~C~ = Control = ~Ctrl~ 键
- ~M~ = Meta = ~Alt~ 键
- ~SPC~ = Space = 空格键
- ~RET~ = Return = 回车键
- ~DEL~ = Delete = 退格键

** 特殊

- ~Esc~ 可用 ~C-[~ 代替

| 按键              | 作用                    | 备注                  |
|-------------------+-------------------------+-----------------------|
| ~C-g~             | 终止命令                | 取消正在编辑的命令    |
| ~C-x C-c~         | 退出 emacs              |                       |
| ~C-u <num> <cmd>~ | 重复 ~<cmd>~ ~<num>~ 次 | 输入多个 ~*~ 比较有用 |

** 移动

- 移动时加上 ~shift~ 进行标记

| 按键        | 作用          | 备注                       |
|-------------+---------------+----------------------------|
| ~M~ / ~C-v~ | 上/下一页     |                            |
| ~C-l~       | 居中当前行    | 多次按使当前行置顶/沉底    |
| ~C-b~ / ~f~ | 前/后一个字符 |                            |
| ~M-b~ / ~f~ | 前/后一个单词 | 以空格区分，会跳过部分标点 |
| ~C-p~ / ~n~ | 上/下一行     |                            |
| ~C-a~ / ~e~ | 行首/尾       |                            |
| ~M-a~ / ~e~ | 句首/尾       | 以句号/分号为界            |
| ~M-<~ / ~>~ | 文件首/尾     | 需要用 ~shift~ 打出来      |

** 编辑

| 按键             | 作用              | 备注                                 |
|------------------+-------------------+--------------------------------------|
| ~C-d~            | 删除下一个字符    |                                      |
| ~M-DEL~ / ~d~    | 剪切上/下一个词   |                                      |
| ~C-k~            | 剪切这一行        | 按一次去掉内容，第二次才会去掉换行符 |
| ~M-k~            | 剪切这一句        |                                      |
| ~C-SPC~ 或 ~C-@~ | 进行标记          | ~C-SPC~ 可能会和输入法冲突           |
| ~C-u C-SPC~      | 跳转到标记        | 多次按会在标记环中循环               |
| ~C-x C-SPC~      | 跳转到全局标记    | 跳转到曾经进入的一系列缓冲区标记     |
| ~C~ / ~M-w~      | 剪切/复制标记部分 |                                      |
| ~C-y~            | 粘贴              | 多次按会被一并粘贴，且可粘贴多次     |
| ~M-y~            | 选择粘贴内容      |                                      |
| ~C-/~            | 撤销              | 按 ~C-g~ 会将操作写入历史，可以 redo |

** 搜索

| 按键           | 作用        | 备注         |
|----------------+-------------+--------------|
| ~C-s~ / ~r~    | 向前/后查找 |              |
| ~RET~ (查找中) | 退出搜索    | 定位到目标点 |

** 文件

| 按键      | 作用     | 备注 |
|-----------+----------+------|
| ~C-x C-f~ | 打开文件 |      |
| ~C-x C-s~ | 保存文件 |      |
| ~C-x C-w~ | 另存文件 |      |

** 缓冲区

| 按键    | 作用       | 备注       |
|---------+------------+------------|
| ~C-x b~ | 切换缓冲区 | 默认最近的 |

** 帮助

- ~C-h~ 可用 ~<f1>~ 代替

| 按键    | 作用         | 备注 |
|---------+--------------+------|
| ~C-h f~ | 搜索命令帮助 |      |
| ~C-h a~ | 搜索所有帮助 |      |
