* emacs 常用快捷键

** 缩写

| 缩写  | 原文    | 按键   |
|-------+---------+--------|
| ~C~   | Control | ~Ctrl~ |
| ~M~   | Meta    | ~Alt~  |
| ~SPC~ | Space   | 空格键 |
| ~RET~ | Return  | 回车键 |
| ~DEL~ | Delete  | 退格键 |

** 特殊

- ~Esc~ 可用 ~C-[~ 代替

| 按键              | 作用                    | 备注                  |
|-------------------+-------------------------+-----------------------|
| ~C-g~             | 终止命令                | 取消正在编辑的命令    |
| ~C-x C-c~         | 退出 emacs              |                       |
| ~C-u <num> <cmd>~ | 重复 ~<cmd>~ ~<num>~ 次 | 输入多个 ~*~ 比较有用 |

** 移动

| 按键        | 作用          | 备注                       |
|-------------+---------------+----------------------------|
| ~M~ / ~C-v~ | 上/下一页     |                            |
| ~C-l~       | 居中当前行    | 多次按使当前行置顶/沉底    |
| ~C-b~ / ~f~ | 前/后一个字符 |                            |
| ~M-b~ / ~f~ | 前/后一个单词 | 以空格区分，会跳过部分标点 |
| ~C-p~ / ~n~ | 上/下一行     |                            |
| ~C-a~ / ~e~ | 行首/尾       |                            |
| ~M-a~ / ~e~ | 句首/尾       | 以句号/分号为界            |
| ~M-<~ / ~>~ | 文件首/尾     | 需要用 ~shift~ 打出来      |

** 编辑

| 按键             | 作用                   | 备注                                 |
|------------------+------------------------+--------------------------------------|
| ~C-d~            | 删除下一个字符         |                                      |
| ~M-DEL~ / ~d~    | 剪切上/下一个词        |                                      |
| ~C-k~            | 剪切这一行             | 按一次去掉内容，第二次才会去掉换行符 |
| ~M-k~            | 剪切这一句             |                                      |
| ~C-SPC~ 或 ~C-@~ | 进行标记               | ~C-SPC~ 可能会和输入法冲突           |
| ~C-u C-SPC~      | 跳转到标记             | 多次按会在标记环中循环               |
| ~C-x C-SPC~      | 跳转到全局标记         | 跳转到曾经进入的一系列缓冲区标记     |
| ~C-x C-x~        | 交换标记与光标并高亮   |                                      |
| ~C~ / ~M-w~      | 剪切/复制标记部分      | 作用范围是标记到光标区域             |
| ~C-y~            | 粘贴                   | 多次按会被一并粘贴，且可粘贴多次     |
| ~M-y~            | 粘贴内容变为上次剪切的 | 在 ~C-y~ 后使用，多次按循环剪切历史  |
| ~C-/~            | 撤销操作               | 按 ~C-g~ 会将操作写入历史，可以 redo |

** 搜索

| 按键           | 作用        | 备注         |
|----------------+-------------+--------------|
| ~C-s~ / ~r~    | 向前/后查找 |              |
| ~RET~ (查找中) | 退出搜索    | 定位到目标点 |

** 文件

| 按键      | 作用     | 备注 |
|-----------+----------+------|
| ~C-x C-f~ | 打开文件 |      |
| ~C-x C-s~ | 保存文件 |      |
| ~C-x C-w~ | 另存文件 |      |

** 缓冲区

| 按键    | 作用       | 备注       |
|---------+------------+------------|
| ~C-x b~ | 切换缓冲区 | 默认最近的 |

** 编程

| 按键  | 作用             | 备注               |
|-------+------------------+--------------------|
| ~M-.~ | 跳转到符号的定义 |                    |
| ~M-,~ | 回到跳转前的文件 |                    |
| ~M-?~ | 查找符号调用     |                    |
| ~M-;~ | 智能添加注释     | 标记行时，注释整行 |

** 命令缓冲区

| 按键        | 作用          | 备注 |
|-------------+---------------+------|
| ~M-p~ / ~n~ | 上/下一个历史 |      |

** 窗口

| 按键        | 作用             | 备注 |
|-------------+------------------+------|
| ~C-x 0~     | 关闭当前窗口     |      |
| ~C-x 1~     | 仅保留当前窗口   |      |
| ~C-x 2~     | 当前窗口上下分裂 |      |
| ~C-x 3~     | 当前窗口横向分裂 |      |
| ~C-x o~     | 切换窗口         |      |
| ~C-M-v~     | 第二窗口翻页     |      |
| ~C-x 4 C-f~ | 在新窗口打开文件 |      |

** 帮助

- ~C-h~ 可用 ~<f1>~ 代替

| 按键    | 作用         | 备注 |
|---------+--------------+------|
| ~C-h f~ | 搜索命令帮助 |      |
| ~C-h a~ | 搜索所有帮助 |      |
